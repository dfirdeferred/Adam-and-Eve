Exchange Vulnerability Enumerator

Command Endpoints

  • /cmd/custom-command
    • send custom command to powershell client. ex: Server-IP/cmd/ipconfig

  • /admins
    • enumerates AD admins

  • /firewall
    • enumerates firewall configuration of client

  • /services
    • retrieves local services

  • /services/dc
    • attempts to retrieve services from Domain Controllers

  • /pwnd
    • endpoint for client to retrieve commands via http

  • /result
    • retieves last result from client

  • /allresults
    • retieves content from the log file

  • /shares
    • enumerates all "server-like computers" for accessible shares

  • /shares/all
    • enumerates all computers for accessible shares

  • /trimarc
    • runs the Trimarc checks script

  • /whoami
    • retrieves data about the client session's current user

  • /dns
    • enumerates domain dns information

  • /portscan
    • performs a port scan of the local subnet

  • /passspray
    • performs password spray against the domain

  • /moab
    • attempts to replicate to other clients

  • /sharphound
    • runs the sharphound collector

  • /azurehound
    • runs the azurehound collector

  • /kill
    • kills all client connections
